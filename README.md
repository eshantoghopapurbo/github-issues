1 Difference between var, let, and const

var: Older way to declare variables. It can be redeclared and is function-scoped.

let: Modern variable declaration. It can be changed but cannot be redeclared in the same scope.

const: Used for values that should not be reassigned after declaration.

2️ What is the spread operator (...)?
The spread operator is used to expand elements of an array or object into individual values. It helps when copying, merging, or passing multiple values easily.

3️ Difference between map(), filter(), and forEach()

map(): Creates a new array by transforming each element.

filter(): Returns a new array containing only elements that match a condition.

forEach(): Runs a function on each element but does not return a new array.

4️ What is an arrow function?
An arrow function is a shorter way to write functions in JavaScript using the => syntax. It makes the code more concise and is commonly used for simple operations.

5️ What are template literals?
Template literals are a way to write strings using backticks (`). They allow variables and expressions to be inserted directly inside the string using ${}.
